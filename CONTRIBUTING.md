# Contributing

¡Gracias por querer contribuir! Este proyecto es open source (MIT) y las PRs son bienvenidas.

## Workflow

1. Haz un fork del repo.
2. Crea una rama descriptiva: `git checkout -b feat/mi-mejora`.
3. Commits pequeños y claros (idealmente en inglés o español consistente).
4. Abre un Pull Request contra `main`.

## Estilo

- **Python**: seguimos PEP 8. Formatea con `black` si puedes.
- **JS/HTML**: 4 espacios, strings simples.
- Sin dependencias nuevas salvo que aporten claramente.

## Reportar bugs

Abre un issue incluyendo:
- URL de PcComponentes con la que falla (si aplica).
- Pasos para reproducir.
- Log de `journalctl -u pccomponentes-tracker` o equivalente.

## Funcionalidades propuestas

Abre un issue etiquetado `enhancement` antes de invertir tiempo — así evitamos trabajo duplicado.
